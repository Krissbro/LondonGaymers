# GiveMe
Allow members to give themselves roles.

## Memes Commands
- `[p]giveme <role>` Gives yourself a role.
- `[p]giveme list` Lists this servers givemes.
- `[p]giveme add <name> <role>` Adds a giveme.
- `[p]giveme remove <name>` Removes a giveme.
